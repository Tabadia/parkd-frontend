# Parkd Frontend

A React-based frontend for **Parkd** â€” a license-plate-based permit verification system. This app provides a user interface for scanning license plates and displaying permit status.

## Usage

1. Open the app in Expo Go
2. Allow camera access when prompted.
3. Capture a photo of the back of a car
4. View permit status
